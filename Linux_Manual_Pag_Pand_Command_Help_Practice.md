# Linux Manual Pages and Command Help Practice

In this lab I practiced using Linux help and documentation commands to learn more about terminal utilities and system commands.

## Commands Used

### Display short command description
```bash
whatis cat
```

### Open the manual page for a command
```bash
man cat
```

### Search manual page descriptions
```bash
apropos -a first part file
```

### Open the manual page for user management
```bash
man useradd
```

## Skills Practiced

- Linux terminal usage
- Using Linux manual pages
- Searching command documentation
- Learning command syntax and options
- Basic Linux system administration
