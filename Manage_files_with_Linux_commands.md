# Linux File and Directory Management Practice

In this lab I practiced creating, removing, moving, and managing files and directories in Linux through the terminal.

## Commands Used

### Create a directory
```bash
mkdir logs
```

### List files and directories
```bash
ls
```

### Remove a directory
```bash
rmdir temp
```

### Navigate to another directory
```bash
cd notes
```

### Move a file to another directory
```bash
mv Q3patches.txt /home/analyst/reports/
```

### Remove a file
```bash
rm tempnotes.txt
```

### Create a new file
```bash
touch tasks.txt
```

## Skills Practiced

- Linux terminal usage
- File and directory management
- Moving and deleting files
- Basic Linux administration
